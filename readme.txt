Версия Java: openjdk version "11.0.28" 2025-07-15

Для запуска util.jar необходимо:
	1. открыть командную строку
	2. перейти в папку с скачанным util.jar файлом
	3. ввести следующее:
		java -jar util.jar -o /some/path -p result- -a -f файл1.txt файл2.txt
		
		Описание опций:
		
			-o // Создает путь для результатов
				Пример -o /some/path
				При введении опции -o необходимо сразу после вводить путь для результатов, иначе следующе стоящий аргумент будет расценен как путь
			
			-p // Задает префикс имен для результатов
				Пример -p result123_
				При введении опции -p необходимо сразу после вводить  желаемый префикс, иначе следующе стоящий аргумент будет расценен как префикс
			-a // Позволяет добавлять к предыдущим результатам нынешние
				Будет искать файл с идентичным названием и местоположением. т.е. если вы хотите к /some/path/res_Integer.txt добавить новые результаты, необходимо указать что новые результаты будут сохранять в /some/path/res_Integer.txt с помощью вышеописанных опций
			-s // Выдает краткое описание результатов в консоль
			-f // Выдает подробное описание результатов в консоль
				В случае если будут введены обе опции -s и -f будет выполняться -f вне зависимости от расположения
				
				Опции возможно вводить и после txt файлов

Сторонние библиотеки не использовались
